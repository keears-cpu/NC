# natal.kr Policy Checklist

PortOne 실연동 또는 운영형 서비스 페이지 준비 전 체크리스트.

## 필수 정책 페이지

- 이용약관
- 개인정보처리방침
- 환불 / 취소 정책

## 페이지에 들어가야 할 핵심 항목

### 이용약관

- 서비스 정의
- 보고서 제공 방식
- 회원/비회원 기준
- 열람 코드 사용 안내
- 책임 제한
- 분쟁 처리 기준

### 개인정보처리방침

- 수집 항목
- 수집 목적
- 보관 기간
- 제3자 제공 여부
- 파기 절차
- 문의처

### 환불 / 취소 정책

- 결제 전
- 결제 후 제작 전
- 제작 시작 후
- 제공 완료 후
- 단순 변심 기준
- 오류 발생 시 처리 기준

## 운영 정보

- 서비스명
- 문의 이메일
- 가능하면 사업자 정보
- 고객 응대 시간

## 결제 페이지에 들어가야 할 안내

- 결제 상품명
- 결제 금액
- 결제 후 제공 방식
- 열람 방식
- 환불 정책 링크
- 개인정보처리방침 링크

## 권장 문구 방향

- 단정적 약속보다 안내형
- 과장된 효능 표현 지양
- 확정 진단처럼 읽히는 문구 지양
- 디지털 리포트 제공 방식 명확화
